## Synopsis

Assignment Code for Analytics for Observational Data (STAT2004)

## Contents
README - README file for assignment code.

Figures - File containing images used in report.

R_Cross_Val - File containing comparisions of Python statistical outputs to R outputs.

exploratory_plots - File containing exploratory plots of League of Legends dataset.

STAT2004 Assignment.ipynb - Jupyter notebook containing code for League of Legends multivariate 
data analysis.

Copula-Usage.Rmd - R markdown file containing a demonstration of modelling multivariate data with a 
copula function.

pipeline_utils.py - Python file containing custom transformer classes and other utility functions for 
use with the sklearn library.

high_diamond_ranked_10min.csv - Data file containing the League of Legends data.

STAT2004 Assignment.pdf - Assignment report

## Dependencies

STAT2004 Assignment.ipynb <- high_diamond_ranked_10min.csv, pipeline_utils.py
